# Array Methods

The declaration for array is given as

```javascript
let arr= new Array()
//or
let arr=[]
```

the above two syntaxes are declaration of arrays.

we use for of loop to iterate through array

```javascript
let arr=[1,2,3]
for(let x of arr){
  console.log(x)
}
```
